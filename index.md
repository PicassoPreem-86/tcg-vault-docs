# TCG Vault — Theme Documentation

Purpose-built theme for Trading Card Game retailers on Shopify. Two presets — **Vault** (dark, singles-first) and **Sealed** (light, sealed+supplies-first) — both driven by the same `tcg.*` metafield namespace and app-block mount points.

## Read in this order

1. [Getting Started](getting-started.html) — install the theme, choose a preset, essential setup
2. [TCG Metafields](tcg-metafields.html) — the `tcg.*` namespace that drives card displays
3. [Product Pages](product-pages.html) — condition variants, TCG product blocks, buylist pricing
4. [Sections Reference](sections.html) — every TCG-specific section, what it does, how to configure it
5. [Game Mega-Menu](mega-menu.html) — setting up the header nav with per-game linklists
6. [Templates & Layouts](templates.html) — the 9 TCG-specific collection/page templates
7. [App Integrations](apps.html) — BinderPOS, TCGplayer Pro, TCG Sync, Crystal Commerce, SortSwift
8. [Preset Differences](presets.html) — Vault vs Sealed: what changes, when to switch
9. [FAQ & Troubleshooting](faq.html) — common questions, gotchas, fixes
