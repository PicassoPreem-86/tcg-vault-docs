# TCG Theme Documentation

Purpose-built theme for Trading Card Game retailers on Shopify. Two presets — **Vault** (dark, singles-first) and **Sealed** (light, sealed+supplies-first) — both driven by the same `tcg.*` metafield namespace and app-block mount points.
