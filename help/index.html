Installation

After purchasing Vault from the Shopify Theme Store:

  • Go to Online Store > Themes in your Shopify admin
  • Click Customize on the Vault theme
  • Choose a preset (Vault, Sealed, Graded, Commander, or Pokemon) from Theme settings > Presets
  • Customize colors, fonts, and sections to match your brand

Presets & Color Schemes

Vault ships with 5 presets, each designed for different TCG store aesthetics:

  • Vault Dark mode with warm gold accents — ideal for singles-focused stores
  • Sealed Light cream palette — optimized for sealed product photography
  • Graded Warm vintage tones — perfect for graded card shops
  • Commander Deep navy with bold blue accents — great for MTG-focused stores
  • Pokemon Clean white with red accents — designed for Pokemon-first inventory

Each preset includes 5 color schemes that can be assigned per-section for visual variety.

Header & Navigation

Announcement Bar

Enable via Header section > Announcement text. Supports a link and custom text (e.g., shipping promotions).

Navigation Menu

Assign your main menu in Header section > Menu. Menu items with children automatically render as dropdown menus. Recommended structure:

  • Games (with children: MTG, Pokemon, Yu-Gi-Oh!, etc.)
  • Singles, Sealed, Preorders, Supplies, Wholesale, Buylist, Events

Predictive Search

The search icon opens a predictive search overlay that shows product results with images and prices as you type.

Homepage Sections

The homepage uses TCG-specific sections that can be reordered in the theme editor:

  • TCG Hero Banner — Full-width hero with background image, heading, and 2 CTA buttons
  • Game Tiles — Colored tile grid linking to game-specific collections (up to 6 games)
  • Specials Banner — Promotional banner with optional countdown timer
  • Pre-order Showcase — Product grid with release dates and countdown timers
  • Singles Grid — Featured singles with TCG badges (set code, rarity, finish)
  • Wholesale Showcase — Case-quantity products for B2B customers
  • Event List — Upcoming events with date, game, format, and RSVP links
  • Supplies Grid — Accessories and supplies product grid

Product Pages

The product page features a two-column layout with 18 block types:

TCG-Specific Blocks

  • Set & Rarity Badge — Displays set code and rarity with color-coded dot
  • Foil/Finish Badge — Shows finish type (Normal, Foil, Etched, Holo) with animated shimmer for foils
  • Condition Price Table — NM/LP/MP pricing in a clean grid
  • Stock Indicator — Per-condition stock counts
  • Card Specs Table — Metafield-driven specifications (artist, card number, color/type)
  • Wholesale Info — Bulk pricing for B2B customers
  • Other Printings — Grid of alternate printings from other sets

Condition Variant Picker

Products with a "Condition" option (NM, LP, MP) display pill-style buttons. Price and stock update dynamically when a condition is selected.

Collection Pages & Filters

Collection pages include faceted filtering with 8 TCG-specific filter groups:

  • Availability, Price, Rarity, TCG Game, Set Code, Finish, Condition, Format Legality

Filters start collapsed to show the product grid immediately. Active filters display as removable pills. Configure filters via Shopify's Search & Discovery app.

Cart & Checkout

Vault supports both cart page and cart drawer (configurable in Theme Settings > Cart type).

Cart Drawer

The slide-out cart drawer includes a free shipping progress bar (configurable threshold), product thumbnails, condition variants, quantity controls, and a checkout button.

TCG-Specific Sections

Decklist Importer

Paste a decklist (one card per line), and the importer matches cards against your inventory, showing prices and an "Add all to cart" button. Supports MTG Arena, MTGO, and plain text formats.

Set Browser

Automatically builds a browsable grid of sets from your product tags. Filter by game (MTG, Pokemon, Yu-Gi-Oh!, One Piece, Lorcana).

Buylist Form

Customer-facing form for selling cards back. Includes payment preference (cash vs store credit), built-in FAQ accordion, and email notification.

Wholesale Application

B2B application form for trade accounts with business details, tax ID, and volume estimates.

TCG Metafields

Vault reads product metafields from the tcg namespace:

  • tcg.game — Game name (MTG, Pokemon, etc.)
  • tcg.set_code — Set abbreviation (e.g., CLU, SV3PT5)
  • tcg.rarity — Card rarity (Common, Uncommon, Rare, Mythic, etc.)
  • tcg.finish — Card finish (Normal, Foil, Etched, Holo)
  • tcg.card_number — Collector number
  • tcg.artist — Card artist name
  • tcg.release_date — Set release date (for preorder countdown)
  • tcg.is_preorder — Boolean flag for preorder products

These metafields power the TCG badges, condition pricing, and set browser. Define them in Settings > Custom data > Products.

Languages & Localization

Vault includes translations for 6 languages: English (default), French, German, Spanish, Italian, and Japanese. All TCG-specific terms are translatable via locale files.
